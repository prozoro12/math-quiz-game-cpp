# Math Quiz Game (C++)

## Description
A console-based math quiz game written in C++ that generates random
math questions and checks user answers.
Uses Windows console for UTF-8 output.

## Features
- Random arithmetic questions
- User input validation
- Score tracking

## Concepts Used
- Variables and data types
- Loops and conditionals
- Random number generation
- Strings

## How to Run
1. Compile `main.cpp` using a C++ compiler
2. Run the generated executable

## Future Improvements
- Difficulty levels
- Timer-based questions
